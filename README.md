# 📝 Flask TO-DO List Web App

A simple and user-friendly TO-DO List web application built with **Flask**, featuring user authentication and task management.

---

## 📌 Features

✅ User Registration & Login  
✅ Secure password hashing  
✅ Session-based authentication  
✅ Add new tasks  
✅ Update task status (Pending → Working → Done)  
✅ Delete individual task  
✅ Clear all tasks (user-specific)  
✅ User-specific task dashboard  
✅ Flash messages for actions  
✅ Responsive UI  

---

## 🛠 Tech Stack

**Frontend:**
- HTML
- CSS

**Backend:**
- Python
- Flask
- Flask-SQLAlchemy

**Database:**
- SQLite
